		O que é e Como utilizar o git e o gitHub - Resumo
---------------------------------------------------------------------------------------
git -> É um controlador de versões de arquivos, 'e diretórios'. Cria um Repositório.

gitHub -> Adiciona os arquivos e diretórios no site, "na Nuvem". Serviço Web.
        Cria um repositório online; público ou privado.
---------------------------------------------------------------------------------------

Comando 1 -> # git init # 
	. Cria um repositório local, na máquina.

Comando 2 -> # git add .#		<Equivalente no ->	VS Code: Preparar atualizações, arquivos.
	. Adicionar todos os arquivos e diretórios do 'diretório raiz' no repositório.

Comando 3 -> # git commit -a -m "mensagem 'Titulo'"# <Equivalente no -> VS Code: Confirmar
	. Adicionar todos os arquivos e diretórios do 'diretório raiz' no repositório.

Comando 4 -> #git remote add origin 'link do repositório' <Equivalente no -> VS Code: Enviar por Push para
	. Adiciona caminho remoto para efetuar o push ..
	. Ex. de link: https://github.com/RamomAndradeMedeirosDaSilva/BackEnd-com-NodeJs

Comando 5 -> # git push origin master#	<Equivalente no -> VS Code: 'Sincronizar Alterações' ou Enviar por Push
	. Adicionar todos os arquivos e diretórios do 'diretório raiz' no repositório.

Comando 1	Comando 2	Comando 3	Comando 4	Comando 5

	Provavelmente é o comando 4
